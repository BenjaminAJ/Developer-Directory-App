# Developer Directory App

A full-stack app to manage developer profiles with role-based filtering and tech stack search.

## Tech Stack
- **Frontend**: React, Tailwind CSS, React Toastify
- **Backend**: Node.js, Express, MongoDB (Mongoose)
- **Validation**: Basic form validation
- **Features**: Add, list, filter by role/tech, responsive UI, toast notifications
